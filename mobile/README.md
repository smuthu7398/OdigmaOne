Phase 1 web APIs are stable (decision: build mobile after Phase 1 so it
targets settled endpoints).

- `@odigma/shared` for API schemas/types
- Theme file generated from `docs/UI_Design_Guide.md` tokens
  (light default, dark toggle)
- Testing: Expo Go on Android phone (scan QR on same WiFi)
